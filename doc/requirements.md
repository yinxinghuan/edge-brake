# 《急刹车》需求文档

## 1. Overview

《急刹车》是一款单手按住刹车、争取让角色停在悬崖边缘最近位置的竖屏短局反应游戏，每局约 20–45 秒。

## 2. Visual Design

游戏采用 390×700 的竖屏 3D 构图，以正交斜俯视镜头呈现立体浮冰、企鹅与下方海面；角色始终是最大视觉焦点，HUD 只显示当前关数、累计分数和连续完美次数。首局通过角色自动滑行、底部按压示意和刹车后的即时姿态变化完成无文字或极少文字教学。完整镜头、灯光、材质、色彩、字体、图标、动效与素材规则写入 `doc/visual.md`。

## 3. Game Mechanics

- 开局角色位于赛道左侧 x=64，初速度为 72 px/s，经过 700 ms 准备时间后开始自动向右移动。
- 每一关的悬崖边缘位于 x=330–350 之间随机变化；角色前缘越过悬崖边缘超过 12 px 即坠落并结束本局。
- 未刹车时加速度为 92 px/s²；第 1 关最高速度为 250 px/s，此后每关增加 12 px/s，最高封顶 370 px/s。
- 玩家按住时施加 310 px/s² 的反向加速度，松开后恢复自然加速；速度不会反向。
- 当速度降至 4 px/s 以下并持续 180 ms，判定停车完成，按角色前缘到悬崖边缘的距离结算：0–8 px 为“贴边”并得 5 分，9–22 px 为“漂亮”并得 3 分，23–52 px 为“安全”并得 1 分，大于 52 px 为“太早”且本关不得分。
- 连续“贴边”形成连击：第 2 次起每次额外获得 `min(连击数 - 1, 5)` 分；非贴边停车会清空连击，但不会结束游戏。
- 成功停车后停留 850 ms 展示距离和评分，然后立即开始下一关；企鹅以阻尼移动回出发位置，避免硬切跳变。
- 刹车时循环显示 9 个低多边形雪屑粒子；贴边停车额外显示 8 枚绕冰崖旋转的低多边形冰晶，持续到 850 ms 结算结束。
- 最佳分数和最高连续贴边次数保存在本地；单局分数从 0 开始。

## 4. Controls

- Pointer / Touch：在游戏区域任意位置按下立即刹车，持续按住则持续制动，抬起立即解除刹车。
- Keyboard：按住 Space 或 Enter 刹车，松开解除；按键自动重复不会重复触发状态切换。
- 开始与重新开始按钮使用单次指针按下；进入游戏后的第一次按压可直接作为刹车输入，不要求额外确认。
- 浏览器失焦、页面隐藏或指针取消时立即解除刹车，避免返回页面后卡在制动状态。

## 5. Win / Lose Conditions

- 游戏没有最终胜利，目标是连续通过更多关并刷新总分与贴边连击纪录。
- 角色前缘越过悬崖边缘超过 12 px 时失败；失败过程持续 650 ms，明确展示滑出、短暂悬停与坠落。
- 结算界面显示本局分数、通过关数、最佳贴边距离、最高连击和历史最高分，并提供“再来一次”主按钮。
- 距离悬崖超过 52 px 的过早停车只记为本关 0 分，850 ms 后继续下一关，不直接结束游戏。

## 6. Sound Effects

- 开始滑行：180–260 Hz 的三角波上滑，时长 100 ms，音量 0.08。
- 按下刹车：135 Hz 锯齿波在 120 ms 内下降至 92 Hz，音量 0.055；每次新的按压触发一次，持续按住不循环堆叠。
- 普通停车：320 Hz 正弦波短音，时长 90 ms，音量 0.10。
- “漂亮”停车：420 Hz 与 620 Hz 两个正弦波依次触发，总时长 180 ms，音量 0.12。
- “贴边”停车：520、780、1040 Hz 三音上行，间隔 55 ms，总时长 260 ms，音量 0.14。
- 坠落失败：160 Hz 锯齿波在 420 ms 内下降至 55 Hz，末尾叠加 80 ms 低频噪声，音量 0.14。
- 界面按钮：600 Hz 正弦波，时长 45 ms，音量 0.06；静音或音频初始化失败时不影响游戏。
