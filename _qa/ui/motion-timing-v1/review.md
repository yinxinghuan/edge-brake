# 动作时序视觉复验

- 目标尺寸：390×700、320×568。
- 检查状态：松手后 120 ms 起步爆发、370 ms 稳定滑行、自动刹车首帧、刹车持续 700 ms、结算。
- 验收标准：起步迈步不延续到 370 ms；滑行仍保留低幅度平衡感；自动刹车在 700 ms 后仍保持；从刹车触发到停止判定至少 1700 ms；两种尺寸无文字遮挡或角色裁切；控制台无错误。

## 结论

通过。390×700 的自动刹车从触发到停车判定为 2304 ms，320×568 为 2101 ms；两个尺寸在刹车开始 700 ms 后仍保持 `data-auto-braking="1"`。刹车帧可读到角色后仰、双臂向后上方抬起、双腿错位外撑及脚后雪屑，未出现角色裁切或 HUD 遮挡。起步帧保留明显的交替迈步；代码中的强步态窗口已由 620 ms 缩短为 360 ms，并在窗口结束后把摆幅压到 8%。370 ms 时镜头已经进入既有的全景拉远过程，角色缩入远景，因此该帧主要用于确认动作阶段与镜头衔接，不作为肢体细节判读依据。控制台错误为 0。
